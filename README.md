## Project Overview

Designed and developed an interactive Power BI dashboard to analyze hospital emergency room operations using patient visit data.

The project transforms raw patient records into operational insights that support monitoring of emergency room performance, patient flow, service quality, and resource utilization.

The analysis focuses on:

- Patient volume and daily trends
- Average wait time
- Patient satisfaction
- Department referrals
- Patient demographics
- Admission patterns

> All data preparation, modeling, DAX measures, and dashboard development were completed in Power BI.

> ## Important Note
> For detailed project documentation, data preparation steps, dashboard design decisions, and business insights, please refer to the [Project Documentation](Hospital%20Emergency%20Room%20Anaysis)

## Business Objectives

- Monitor emergency room patient volume over time.
- Evaluate patient wait time performance.
- Measure patient satisfaction trends.
- Analyze referral patterns across hospital departments.
- Identify demographic and admission trends to support operational planning.

## Data & Modeling

The solution was built using a star schema consisting of:

- A centralized **Date** dimension supporting time intelligence.
- A single **Hospital ER** fact table containing patient visit records.
- Additional calculated fields to classify admissions, age groups, wait time performance, and reporting periods.

The model enables dynamic filtering across months and years while maintaining a single consolidated dashboard experience.

## Key Insights

The dashboard highlights fluctuations in emergency room demand, identifies peak operating periods, and monitors patient wait time performance against service targets.

It also provides visibility into referral patterns, admission rates, and demographic distributions, enabling stakeholders to better understand patient flow and support data-driven operational improvements.
